Detailed Description

This struct encapsulates various application state and configuration settings. The fields of the struct roughly correspond to what one normally would have defined as data members in a class-based object oriented language setting. However, since a C struct cannot have methods, we allocate mbs on the stack in main, and pass it explicitly, as an argument, to the various functions that require access to application state.
